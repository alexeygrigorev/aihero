## 🎯 Course Overview

This intensive 8-week bootcamp is designed for developers, data scientists, and AI enthusiasts who want to build real-world AI applications. You'll progress from understanding basic RAG systems to creating intelligent agents that can use tools, make decisions, and interact with external systems.

### What You'll Build
- **RAG Systems**: Document search and question-answering applications
- **AI Agents**: Intelligent systems that can use tools and make autonomous decisions  
- **Agent Orchestration**: Multi-agent systems with handoffs and coordination
- **Production Systems**: Scalable, deployed AI applications

## 📚 Weekly Syllabus

### **Week 1: Introduction to RAG**
- RAG fundamentals and architecture
- Vector databases and embeddings
- Building your first Q&A system
- Evaluation and optimization techniques

### **Week 2: Advanced RAG Techniques**
- Hybrid search strategies
- Context retrieval optimization
- Multi-document RAG systems
- RAG evaluation frameworks

### **Week 3: Introduction to Agents**
- Agent architecture and concepts  
- Function calling and tool use
- Building simple task-oriented agents
- Agent memory and state management

### **Week 4: Agent Tools and Integration**
- Creating custom tools for agents
- External API integration
- File system operations
- Database interactions

### **Week 5: Multi-Agent Systems**
- Agent orchestration patterns
- Agent handoffs and communication
- Collaborative problem solving
- Conflict resolution strategies

### **Week 6: Production Agent Development**
- Agent monitoring and logging
- Error handling and recovery
- Performance optimization
- Security considerations

### **Week 7: Advanced Agent Frameworks**
- OpenAI Agents SDK
- PydanticAI implementation
- LangChain agent patterns
- Custom framework development

### **Week 8: Deployment and Scaling**
- Production deployment strategies
- Model Context Protocol (MCP)
- Monitoring and observability
- Final project presentations

### **Course Structure & Format**

**Q: How is the course delivered?**
A: Live online sessions with hands-on coding, recorded for later review. Each week includes 2-3 hours of live instruction plus self-paced assignments.

**Q: What if I miss a live session?**
A: All sessions are recorded and available within 24 hours. You'll also receive session notes and code repositories.

**Q: How much time should I dedicate weekly?**
A: Plan for 6-8 hours per week: 2-3 hours live sessions + 3-5 hours for assignments and practice.

**Q: Do I need GPU access for this course?**
A: No GPU required! We use cloud APIs for LLM inference. Standard laptop/desktop is sufficient.
